This application enables the user to:

1) Add a doctor to the database through entering their personal data.

2) Search for doctor using their ID

3) Delete a doctor from the database by inserting their ID

4) Delete a patient (an appointment) from the database

5) Search for a patient using their ID

6) Display all available doctors

7) Checks the availablility of a doctor in a boolean function

7) Display all of today's patients

8) Create appointments that are very efficient, as they go chronologically by filling every hour first before moving on to next hours. No doctor can have more than one appointment per hour. The program simulates a working day from 9am to 11pm (23:00)
